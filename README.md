# EchoTome-WoW

EchoTome 是魔兽世界正式服轻量聊天记录插件，在本地自动保存所有聊天记录，支持随时翻看、关键词搜索、一键复制。

## ✨ 功能特性
- **全面记录**：支持私聊、战网私信、公会/小队/团队/副本/战场、频道、大喊/表情等所有聊天类型
- **本地存档**：按服务器、角色独立存储，不影响游戏内聊天
- **灵活过滤**：可设置关键词过滤，仅不存档，不屏蔽消息
- **双策略清理**：公共聊天/私聊可分别设置「最多条数」「最长保留天数」，到期自动清理
- **高级浏览器**：会话列表、关键词搜索、日期筛选、纯文本复制、职业配色、物品链接悬停查看
- **自定义提示**：私聊/战网私信支持自定义提示音
- **轻量化**：低占用，加载快

## 📦 安装说明
1.  将插件文件夹放入 `World of Warcraft/_retail_/Interface/AddOns/` 目录
2.  进入游戏即可生效
